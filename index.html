Have you ever tried to follow a web comic, but were not really sure what was referenced from 100 panels back? Ever wonder where that supporting character went off too? What about your favorite quote, what strip was that in?

We want to build a web app that can be used to allow users to comment on stories, point out plot lines, tag characters, and track all the amazing stories that are created along the way. This will allow for a story board experience, being able to link comics backwards and forwards on user defined content.

We need your help. We think this project will be useful to everyone and will release it as open source. To get started we need $10,000 USD to build this application. This will go towards initial hardware costs and paying for 1 developer for 3 months. Once that time period is up we hope the community will continue to develop this software.

We will be licensing this under MIT for final release, so the community can grow it further.
